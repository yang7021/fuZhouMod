# 关于面具能力牌的使用、激发、销毁设计

1. 最多只能持有两个不同面具能力效果，同类型面具重复使用可效果叠加，使用【塔拉面具】增加槽位的功能不可叠加，最多就是持有 2+2 或 2+3 个不同面具的能力效果。
2. 用栈记录面具的使用顺序，比如先用A面具，再用B面具，此时已经到了两个不同面具的上限，此时可以不限制继续使用A和B面具；
3. 此时又用了A面具，这样有两层A，一层B，使用顺序依然是A(2)-B(1)，因为A是最先的。当再用C面具时，两层A面具被【销毁】，此时使用顺序为B(1)->C(1)。
4. A面具被【销毁】的时候，每有一层A就触发一次A的效果，将对应的【黑影兵团】牌加入手牌，由于A有两层那么总共触发两次A
5. 此时顺序是B(1)->C(1)，当再使用D面具时，B被【销毁】，B只有一层，那么B被触发一次。
6. 关于【销毁】时的触发和回合开始时的自动触发不矛盾，是两个独立的触发场景

# 关于黑影兵团卡

- 所有的黑影军团卡都为消耗卡，防止回合累加，卡组里的军团卡越堆越多。
