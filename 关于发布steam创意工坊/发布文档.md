太棒了！完成一个《杀戮尖塔》（Slay the Spire）的 Mod 是一件非常有成就感的事情，恭喜你走到发布这一步！

既然你已经准备好了 `.jar` 包、描述信息和图片，接下来就可以使用游戏官方自带的工具直接将它发布到 Steam 创意工坊。以下是标准且最稳妥的发布流程：

### 第一步：找到官方上传工具
《杀戮尖塔》的安装目录中自带了一个名为 `mod-uploader.jar` 的官方上传工具。
1. 打开 Steam，在库中右键点击“Slay the Spire” -> **管理** -> **浏览本地文件**。
2. 在打开的游戏根目录中，找到 `mod-uploader.jar` 这个文件。

### 第二步：创建 Mod 工作区
你需要在这个目录下使用命令行来创建一个专门用于上传的文件夹（工作区）。
1. 在游戏根目录的文件夹路径地址栏中直接输入 `cmd` 并回车，这会在当前路径下打开命令提示符。
2. 输入以下命令并执行（将 `YourModName` 替换为你的 Mod 英文缩写或代号，建议不要有空格）：
   `java -jar mod-uploader.jar new -w YourModName`
3. 执行后，游戏目录下会生成一个新的文件夹，名字就是你刚才输入的 `YourModName`。

### 第三步：放入文件与封面图
进入刚刚创建的 `YourModName` 文件夹，你会看到一个名为 `content` 的子文件夹和一个 `config.json` 文件。
* **放入 Jar 包**：将你编译好的 Mod `.jar` 文件放进 `content` 文件夹中（玩家下载后实际加载的就是这个文件夹里的内容）。
* **准备封面图**：Steam 创意工坊需要一张封面预览图。挑一张你准备好的高质量图片（建议尺寸为正方形，文件大小**务必小于 1MB**），将其重命名为 `preview.png`（或 `.jpg`），放在与 `config.json` 同级的目录下。

### 第四步：填写配置与描述信息
使用文本编辑器（如记事本或 VS Code）打开 `config.json` 文件。这是你 Mod 的“名片”，你需要将准备好的信息填入其中：
* **Title**：填入你希望在创意工坊展示的 Mod 名称。
* **Description**：填入你的描述信息。如果你有换行需求，可以使用 `\n`。
* **Tags**：填写相关的标签，比如 `Character`, `Cards`, `Relics` 等，方便玩家搜索。

封面图就直接替换 *image.jpg* 即可

> **注意：** 在进行接下来的上传步骤前，请确保你的 **Steam 客户端正在后台运行**，并且已经登录了你想要发布该 Mod 的账号。

### 第五步：一键上传
确认所有文件就位、信息填写无误后，回到刚才的命令提示符窗口（确保路径依然是游戏根目录），输入以下命令：
`java -jar mod-uploader.jar upload -w YourModName`

```shell
# 可能会乱码
java -jar mod-uploader.jar upload -w 圣主历险记

# 强制使用UTF-8
java -Dfile.encoding=UTF-8 -jar mod-uploader.jar upload -w 圣主历险记
```

工具会自动打包并与 Steam 通信。等待几秒钟，如果命令行提示成功，你的 Mod 就已经发布到 Steam 创意工坊上了！

刚上传的 Mod 默认可见性通常是“隐藏”或“开发者可见”。你可以前往 Steam -> 你的个人资料 -> **创意工坊物品** 中找到它，上传更多你在游戏内的截图，检查排版，确认一切完美后，再将可见性修改为“公开”。
